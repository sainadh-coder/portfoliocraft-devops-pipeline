how it ships

One push, five stages, zero manual steps.

Every commit to this repository walks the same path: Jenkins pulls the code, runs the checks, builds the image, and rolls it out — while Graphite and Nagios watch the result in real time.

The assignment target is a working DevOps loop, not just a website — so the pipeline itself is treated as the primary deliverable, with the site as its running proof.
